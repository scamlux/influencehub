## Важно про верификацию
`tsc` (typecheck) **стабильно виснет в этой среде** (0% CPU, не завершается — параллельный dev-сервер/песочница). Поэтому весь TS-код проверен **транспиляцией через esbuild** (синтаксис, импорты, JSX, JSX) + ручным ревью типов. Перед мержем прогнать `npm run typecheck && npm test` в нормальном окружении.
